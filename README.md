Create the database:
python -m dbutils

Start the web server:
python -m appserver

Activate the virtual environment: (ON THE CLIENT SIDE!)
venv\Scripts\activate (Windows)
source venv/bin/activate (Mac/Linux)

npm run dev on client folder!
